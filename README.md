# Piedra-Tiejeras-Papel
Esta app es un minijuego de piedra papel o tijeras para perder el tiempo un rato.

Instrucciones para descargar y correr el programa
1. Descargar desde GitHub
Entra al repositorio del proyecto en GitHub.
https://github.com/TortillaDePollo/Piedra-Tiejeras-Papel/blob/main/Piedra-Papel-Tijeras.py
2. Da clic en el botón “Code” y selecciona “Download ZIP”.
3. Descomprime el archivo en tu computadora y abre la carpeta del proyecto.
4. Descargar solo el archivo .py
5. GitHub, abre el archivo piedra_papel_tijeras.py y da clic en el botón “Raw”.
6. Presiona Ctrl + S y guárdalo en tu computadora.
