# Piedra-Tiejeras-Papel
Esta app es un minijuego de piedra papel o tijeras para perder el tiempo un rato.

Instrucciones para descargar y correr el programa
1. Descargar desde GitHub
Entra al repositorio del proyecto en GitHub.
(Ejemplo: https://github.com/usuario/piedra-papel-tijeras
)

Da clic en el botón “Code”.

Selecciona “Download ZIP”.

Descomprime el archivo en tu computadora.

Abre la carpeta del proyecto.

Verifica que el archivo piedra_papel_tijeras.py esté dentro.

✔ Opción 2: Descargar solo el archivo .py

En GitHub, abre el archivo piedra_papel_tijeras.py.

Da clic en el botón “Raw”.

Presiona Ctrl + S y guárdalo en tu computadora.

▶ Cómo ejecutar el programa (Windows, Mac o Linux)
1. Instalar Python (si no lo tienes)

Entra a: https://www.python.org/downloads

Descarga Python 3.10 o superior.

Instálalo marcando la casilla “Add Python to PATH”.

2. Abrir una terminal

Windows:
Presiona Win + R, escribe cmd y presiona Enter.

Mac:
Abre Terminal desde Aplicaciones.

Linux:
Abre tu terminal habitual.
